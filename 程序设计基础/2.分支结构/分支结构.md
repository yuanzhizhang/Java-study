## 布尔表达式

> 布尔表达式的结果只有两种：true（1）和false（0）

**关系操作符：**

| 操作符 | 名称     | 示例    | 结果  |
| ------ | -------- | ------- | ----- |
| <      | 小于     | 10 < 0  | false |
| <=     | 小于等于 | 10 <= 0 | false |
| >      | 大于     | 10 > 0  | true  |
| >=     | 大于小于 | 10 >= 0 | true  |
| ==     | 等于     | 10 == 0 | false |
| !=     | 不等于   | 10 != 0 | true  |

## 逻辑操作符 <sup>[真值表](https://zh.wikipedia.org/wiki/逻辑运算符)</sup>

| 操作符 | 名称 | 说明                                       | 示例            |
| ------ | ---- | ------------------------------------------ | --------------- |
| ！     | 非   | 逻辑非（真的为假，假的为真）               | !false  (真)    |
| &&     | 与   | 逻辑与（两个同时为真时结果为真，否则为假） | 1 && 0 （假）   |
| \|\|   | 或   | 逻辑或（同时为假时结果为假，否则为真）     | 1 \|\| 0 （真） |
| ^      | 异或 | 逻辑异或（一个为真一个为假时结果为真）     | 1 ^ 0 （真）    |
